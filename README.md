 
 # emit-cli
 
 > A template generation cli for all languages and frameworks.

 - Pull Requests are Invited
 
 ## Installation
 
 ```
 npm i emit-cli -g
 ```
 
 ## Usage
 
 ```
 emit [filetype]... [filename]... [framework]...
 ```
 
 For Example
 * with using a framework
 
 ```
 emit html unicorn
 ```
 > Note : default name is ```Index```  

 * with using a framework

```
 emit js Simba --rfce
 ```

 ## Help

For Help use this command
```
emit --help
```
